# Angular Hands-on — Student Course Portal

A standalone-component Angular app demonstrating components, data binding,
structural directives (`*ngFor`, `*ngIf`), event binding, a service, and
dependency injection — the base of the incrementally-built Student Course Portal.

`node_modules` is intentionally excluded (see submission guidelines).

## Run
```bash
cd student-course-portal
npm install
npm start
```
