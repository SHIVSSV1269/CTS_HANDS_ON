# ReactJS Hands-on

A Vite + React single-page app (`reactjs-hol`) with one component per mandatory lab:

| Lab | Concept | Component |
|-----|---------|-----------|
| 2 & 3 | Class vs function components | `ClassComponent`, `FunctionComponent` |
| 4 | Component lifecycle (`componentDidMount`) | `LifecycleDemo` |
| 5 | Styling (CSS Modules + inline) | `StyledComponent` |
| 9 | ES6 features | `Es6Demo` |
| 10 | JSX / `React.createElement` | `JsxDemo` |
| 11 | Events & synthetic events | `EventsDemo` |
| 12 & 13 | Conditional rendering, lists & keys | `ConditionalAndList` |

## Run
```bash
cd reactjs-hol
npm install
npm run dev
```
