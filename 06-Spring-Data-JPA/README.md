# Spring Data JPA with Hibernate — orm-learn

Spring Boot project demonstrating the Spring Data JPA quick example with a
`Country` entity, `CountryRepository extends JpaRepository`, and a `CountryService`.

Runs on an **H2 in-memory database** out of the box (sample data in `data.sql`).
MySQL configuration is included (commented) in `application.properties`.

See `../JPA_vs_Hibernate_vs_SpringDataJPA.md` for the concepts write-up.

## Run
```bash
mvn spring-boot:run
```
